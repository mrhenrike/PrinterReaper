üöÄ Usage

Discover Printers

# Automatic SNMP discovery
python3 printer-reaper.py

# Output:
Discovered printers:
192.168.1.100    HP LaserJet 4250
192.168.1.105    Brother MFC-7860DW

Connect to Printer

# Auto-detect best language
python3 printer-reaper.py 192.168.1.100 auto

# Or specify language manually
python3 printer-reaper.py 192.168.1.100 pjl   # PJL mode
python3 printer-reaper.py 192.168.1.100 ps    # PostScript
python3 printer-reaper.py 192.168.1.100 pcl   # PCL

Execute Commands

192.168.1.100:/> id              # Identify printer
192.168.1.100:/> ls              # List files
192.168.1.100:/> network         # Get network info
192.168.1.100:/> download /etc/passwd    # Download file
192.168.1.100:/> help upload     # Get help on command

Key Commands

  • id - Identify printer model and firmware
  • ls - List directory contents
  • download - Download files from printer
  • upload - Upload files to printer
  • network - Show network configuration
  • nvram - Access NVRAM (PJL)
  • enumerate_operators - Test PostScript operators
  • payload - Execute attack payloads
  • capture - Capture print jobs
  • mirror - Mirror entire filesystem
